‚úÖ Leetcode 236: Lowest Common Ancestor of a Binary Tree ‚Äì Java Solution (0ms, O(n) Time)

class Solution {
  public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    if (root == null || root == p || root == q)
      return root;

    TreeNode left = lowestCommonAncestor(root.left, p, q);
    TreeNode right = lowestCommonAncestor(root.right, p, q);

    if (left != null && right != null)
      return root;

    return left != null ? left : right;
  }
}

‚Üí Base case: agar root null hai ya root hi p ya q hai
‚Üí Toh wahi node ko return kar do (kyunki mil gaya ya tree khatam ho gaya)

return root;
‚Üí Agar base case true tha, toh root return kar diya (null, p ya q)

‚Üí Left subtree me LCA dhoond rahe hain
‚Üí Agar p ya q ya LCA left subtree me mila toh left me store hoga


TreeNode right = lowestCommonAncestor(root.right, p, q);
‚Üí Right subtree me bhi same logic lagaya hai
‚Üí Agar p ya q ya LCA right subtree me mila toh right me store hoga


if (left != null && right != null)
‚Üí Agar dono side se result mila, matlab ek node left me hai, doosra right me
‚Üí Toh current node hi Lowest Common Ancestor hoga


return root;
‚Üí Dono branches me p aur q mile, toh root hi LCA return karo

return left != null ? left : right;
‚Üí Agar dono branches me nahi mila, toh jo branch se mila wahi return karo
‚Üí Agar left non-null hai toh left return karo, warna right
